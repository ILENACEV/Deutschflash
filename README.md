# ⚡ DeutschFlash – Паметно учење германски

[![Next.js](https://img.shields.io/badge/Next.js-15-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?style=for-the-badge&logo=typescript)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-3-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)
[![PWA](https://img.shields.io/badge/PWA-Ready-orange?style=for-the-badge)](https://web.dev/progressive-web-apps/)

**DeutschFlash** е врвна едукативна платформа за учење германски јазик, специјално дизајнирана за македонски говорители. Користиме научни методи, како **SM-2 Spaced Repetition** алгоритмот, за да ви овозможиме трајно паметење на вокабуларот со минимален напор.

---

## 🚀 Клучни Карактеристики

*   **📱 Офлајн-прво искуство (PWA)**: Инсталирајте ја апликацијата на вашиот телефон или компјутер и учете каде и да сте, дури и без интернет конекција.
*   **🧠 SM-2 Алгоритам**: Паметен систем кој знае точно кога треба да го повторите секој збор пред да го заборавите.
*   **🔊 Премиум Voice Engine**: Најприроден изговор на германски јазик со интелигентно кеширање и поддршка за сите прелистувачи (Chrome, Safari, Edge).
*   **🎮 Игри и Вежби**: Од *VocaSwipe* до *Story Mode*, учењето е забавно и интерактивно.
*   **📈 Детална Статистика**: Следете го вашиот напредок, streak и XP поени во реално време.
*   **🔒 Приватност на прво место**: Сите ваши податоци се чуваат локално на вашиот уред (IndexedDB). Нема регистрација, нема следење.

---

## 🛠 Технолошки Стек

Платформата е изградена со најсовремените веб технологии:

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **Држање податоци**: IndexedDB (преку `idb`) за офлајн капацитет од 100%.
- **Анимации**: Framer Motion & Tailwind Animate.
- **Икони**: Lucide React.
- **Инфраструктура**: Vercel & Vercel Analytics.

## 📦 Брза Инсталација

Доколку сакате да го стартувате проектот локално:

1. **Клонирајте го репозиториумот**:
   ```bash
   git clone https://github.com/ILENACEv/Deutschflash.git
   cd Deutschflash
   ```

2. **Инсталирајте ги зависностите**:
   ```bash
   npm install
   ```

3. **Стартувајте го развојниот сервер**:
   ```bash
   npm run dev
   ```

4. Отворете го [http://localhost:3000](http://localhost:3000) во вашиот прелистувач.
