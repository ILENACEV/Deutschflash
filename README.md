## 1. Преглед на Платформата

### 1.1 Што е DeutschFlash?

**DeutschFlash** е напредна едукативна платформа за учење германски јазик, специјално дизајнирана за македонски говорители. Платформата користи математички модели за меморирање и напредни методи на гејмификација за да го направи учењето ефикасно и забавно.

### 1.2 Клучни Карактеристики

| Карактеристика | Опис |
|---------------|------|
| **100% Офлајн** | Работи без интернет конекција |
| **Local-First** | Сите податоци се чуваат локално |
| **SM-2 Алгоритам** | Напредно меморирање со spaced repetition |
| **PWA** | Може да се инсталира како апликација |
| **Гејмификација** | XP, нивоа, streak, одликија |

## 3. Систем за Меморирање (SM-2 Алгоритам)

### 3.1 Што е SM-2?

SM-2 (SuperMemo 2) е математички алгоритам за spaced repetition кој го оптимизира времето помеѓу прегледите на секој збор. Алгоритмот го измислил Пјотр Вожниек (Полска) во 1987 година.

### 3.2 Имплементација во DeutschFlash

**Логиката е во `src/lib/sr-logic.ts`:**

```typescript
function calculateSpacedRepetition(word: VocabularyWord, known: boolean) {
  // Ако е точен одговор:
  // - new → learning (1 ден)
  // - learning → learning (6 дена)
  // - learning → learned (интервал * easeFactor)
  
  // Ако е неточен одговор:
  // - Се враќа на "difficult" статус
  // - Интервалот се ресетира на 1 ден
}
```

### 3.3 Клучни Параметри

| Параметар | Опис | Почетна вредност |
|-----------|------|------------------|
| `interval` | Денови до следен преглед | 0 |
| `easeFactor` | Леснотија на зборот | 2.5 |
| `status` | new / learning / learned / difficult | new |
| `nextReview` | Timestamp за следен преглед | - |

### 3.4 Сортирање на Сесијата

Зборовите се сортираат по приоритет:
1. **Доспеани зборови** (due) - најстарите прво
2. **Нови зборови** - случаен редослед

## 4. Локално Складирање (IndexedDB)

### 4.1 База на Податоци

DeutschFlash користи **IndexedDB** преку `idb` библиотеката за чување на сите податоци во прелистувачот.

**Schema (`src/lib/db.ts`):**

```typescript
interface DeutschFlashDB {
  vocabulary: {
    key: string;           // word ID
    value: VocabularyWord;
    indexes: { 'by-status': string };
  };
  stats: {
    key: string;           // 'userStats' | 'settings' | 'sessions'
    value: UserStats | { sessionSize: number } | LearningSession[];
  };
  verbs: { key: number; value: VerbData; };
  adjectives: { key: number; value: AdjectiveQuestion; };
  idioms: { key: number; value: Idiom; };
  cases: { key: number; value: CaseQuestion; };
}
```

### 4.2 Податоци што се чуваат

| Store | Опис | Пример |
|-------|------|--------|
| `vocabulary` | Сите зборови со SM-2 податоци | ~3000+ |
| `stats` | Кориснички статистики | XP, streak |
| `settings` | Подесувања | sessionSize |
| `sessions` | Историја на сесии | 100+ записи |
| `verbs` | Глаголи за вежба | 100+ |
| `adjectives` | Придавки за вежба | 50+ |
| `idioms` | Идиоми | 30+ |
| `cases` | Падежни вежби | 50+ |

### 4.3 Бекап и Увоз

**Експорт (`/settings`):**
```typescript
storage.exportFullBackup() // → JSON фајл
```

**Увоз:**
```typescript
storage.importFullBackup(jsonString) // → Boolean
```

## 6. Едукативни Модули

### 6.1 VocaSwipe (Главна Сесија)

**Опис:** Tinder-style картички за учење вокабулар

**Функционалност:**
- Swipe right = Знаеш го зборот
- Swipe left = Треба да го повториш
- Tap = Слушај изговор (TTS)
- SM-2 автоматско сортирање

**Интерфејс:**
```
┌─────────────────────────────┐
│  🐕 der Hund               │
│                             │
│  "Der Hund ist groß."       │
│  Кучеeto e големо.         │
│                             │
│  [🔊]  [←]        [→]      │
└─────────────────────────────┘
```

### 6.2 Германски Спелувач

**Опис:** Правописна вежба - составување збор од букви

**Пример:**
```
Досјо: "куче" → _ _ _ _ _
Корисник: H-U-N-D
Точно! ✅
```

### 6.3 Слухо-Профи

**Опис:** Тренинг за препознавање преку слух

**Функционалност:**
- TTS го читае зборот
- Корисникот го избира точниот одговор
- Тренира аудио препознавање

### 6.4 Story Mode

**Опис:** Интерактивни приказни на германски

**Нивоа:** A1, A2, B1

**Пример приказни:**
- "Фриц во Берлин"
- "Ана на одмор"
- "Децата во училиште"

### 6.5 Градител на Реченици

**Опис:** Вежба за редослед на зборови

**Вежби:**
- V2 правило (Verb Second)
- Nebensatz ( субordinирани реченици)
- Rедослед на зборови во германски

### 6.6 Падежен Мајстор

**Опис:** Вежби за германски падежи

**Падежи:**
- **Nominativ** (Именувален) - субjект
- **Akkusativ** (Акузатив) - директен обjект
- **Dativ** (Датив) - индиректен обjект
- **Genitiv** (Генитив) - припадба

### 6.7 Аџектив-Мастер

**Опис:** Деклинација на придавки

**Вежби:**
- Придавки со определен член
- Придавки со неопределен член
- Придавки без член

### 6.8 Глаголски Мајстор

**Опис:** Тренинг за глаголски времиња

**Времиња:**
- **Präsens** (Сегашно)
- **Perfekt** (Перфект)
- **Präteritum** (Идно)
- **Futur I** (Идно)

### 6.9 Дијалози

**Опис:** Реални разговори за секојдневни ситуации

**Теми:**
- Во ресторан
- На доктор
- Во продавница
- На аеродром
- Во хотел
- На работно место

### 6.10 Фрази за Преживување

**Опис:** Најважните фрази за туристи и патници

**Теми:**
- Патување
- Туризам
- Работа
- Здравје
- Купување
- Социјални
- Хоби
- Училиште
- Љубов
- Свадба

## 7. Гејмификација

### 7.1 XP Систем

**Освојување на XP:**
| Акција | XP |
|--------|-----|
| Точен одговор | +10 |
| Завршена сесија | +50 |
| Streak ден | +25 |

### 7.2 Нивоа

**Пресметка:**
```typescript
level = Math.floor(Math.sqrt(xp / 100)) + 1
```

**Пример:**
- Level 1: 0 XP
- Level 2: 100 XP
- Level 3: 400 XP
- Level 4: 900 XP

### 7.3 Streak (Серија)

- Брои колку последователни дена учиш
- Се ресетира ако пропуштиш ден
- Прикажан на dashboard со 🔥 иконата

### 7.4 Одликија (Achievements)

| ID | Наслов | Опис | Услов |
|----|--------|------|-------|
| streak_5 | Фанатик | Одржувај 5 дена streak | streak ≥ 5 |
| learned_50 | Полиглот во најава | Научи 50 зборови | learned ≥ 50 |
| case_master | Падежен Мајстор | Освој над 150 поени во Падежи | highScoreCases ≥ 150 |
| verb_master | Глаголски Експерт | Освој над 200 поени во Глаголски | highScoreVerbs ≥ 200 |
