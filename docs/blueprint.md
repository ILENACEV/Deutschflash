# **App Name**: VocaSwipe Deutsch

## Core Features:

- Swipeable Vocabulary Cards: Displays German vocabulary cards with German word, Macedonian translation, German example sentence, Macedonian sentence translation, and an illustrative visual cue. Users swipe right to mark as known, or left to mark for review.
- Card Reveal Interaction: Allows users to tap or flip cards to reveal the Macedonian translation of the word and the example sentence, mimicking the Tinder card interaction.
- Intelligent Spaced Repetition: An algorithm that prioritizes new words, difficult words (swiped left), and schedules periodic reviews of known words to optimize long-term memory retention. Adjusts word frequency based on user performance.
- Session Configuration: Enables users to select daily learning session sizes of 5, 10, 15, or 20 new words.
- Vocabulary Import Tool: Supports importing new German-Macedonian vocabulary sets via JSON, allowing users to easily add their custom learning material.
- Contextual Image Generation Tool: Automatically generates or suggests a relevant illustrative image or emoji for each vocabulary card based on the German word or sentence, leveraging generative AI to enhance visual memory.
- Learning Progress Tracking: Displays current session progress, basic statistics like total learned words, and streaks to motivate continuous learning.

## Style Guidelines:

- Primary color: A rich, calming forest green (#238F47) to symbolize growth and natural learning progression in a light color scheme.
- Background color: A very light, desaturated green-grey (#E9F3EB) providing a clean, unobtrusive canvas that aids focus and reduces eye strain.
- Accent color: A vibrant lime green (#68E033) for calls to action, highlights, and interactive elements, offering an energetic contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif for its modern, clean, and highly readable qualities across all text elements on mobile.
- Use minimalist, clean line icons for navigation and actions, ensuring visual clarity and avoiding clutter on cards and menus.
- Implement a 'clean mobile interface' focusing on ample white space and clear information hierarchy, optimizing for quick glance and intuitive interaction.
- Smooth Tinder-like swipe animations for vocabulary cards and a subtle card-flip animation for revealing translations, enhancing user engagement and interactivity.
